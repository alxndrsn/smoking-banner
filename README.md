Smoking Banner
==============

Modern browser extension to apply UK government tobacco warnings to social media sites.

Ref: https://www.legislation.gov.uk/uksi/2016/507/contents

# Supported sites

See [`src/manifest.json`](src/manifest.json).

# Development

See [`Makefile`](Makefile) for available targets.

## Setup

On first-run, you may need to change some Chromium settings:

* enable dev mode
* explicitly enable the extension
